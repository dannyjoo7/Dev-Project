# Dev-Project

개인 학습 및 취미 목적으로 개발한 프로젝트 모음입니다.

## Android

| 프로젝트 | 설명 | 기술 스택 |
|----------|------|-----------|
| [**Miruni**](./android/Miruni) | 스마트폰 잠금 해제 시 할 일과 일정을 표시해주는 개인 비서 앱 | Kotlin, Jetpack Compose, Clean Architecture, Hilt, Room, Flow |
| [**WooDong2**](./android/WooDong2) | GPS 기반 지역 커뮤니티 앱 (게시글, 모임, 실시간 채팅) | Kotlin, Firebase, MVVM, Naver/Kakao Map API |
| [**MyMedia**](./android/MyMedia) | YouTube API 기반 영상 검색 및 스트리밍 앱 | Kotlin, Retrofit, ViewPager2, Coroutines, YouTube Player API |
| [**SearchImage**](./android/SearchImage) | Kakao API를 이용한 이미지/동영상 검색 앱 | Kotlin, Retrofit, MVVM, Coroutines |
| [**android_authApp**](./android/android_authApp) | 로그인/회원가입 인증 앱 | Kotlin, Jetpack Compose, Material3 |
| [**nbcmap-todo-list**](./android/nbcmap-todo-list) | 할 일 목록 관리 앱 | Kotlin, ViewBinding, Material |
